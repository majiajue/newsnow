   - `/api/news/cls/detail.get.ts`：获取财联社文章详情
   - `/api/news/cls/jinjia.post.ts`：使用金佳 API 获取文章详情并生成摘要

2. **前端组件**
   - `ClsJinjiaNewsSummary.tsx`：展示文章摘要和评论
   - `useClsJinjiaSummary.ts`：处理与后端 API 交互的 Hook

3. **路由**
